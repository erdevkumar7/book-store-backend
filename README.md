Project: Book-store

Description: 
Book-store is basically online bookstore application where 
users can browse, search for books, view book details, and add books to their cart for 
purchase

Steps to run the project
1. Clone from github
2. Install dependencies: npm install
3. Project run command: npm run dev

Installed dependencies: 
1. express (Version- 4.18.2)
2. bcrypt (version- 5.1.1)
3. body-parser (Version- 1.20.2)
4. cors (Version - 2.8.5)
5. dotenv (Version- 16.4.5)
6. jsonwebtoken (Version- 9.0.2)
7. mongoose (Version - 8.1.3)
8. nodemon(as devdependency) (Version- 3.0.3)

Frontend:
react.js (Version- ^18)
next.js (Version- 14.1.0)

Database: mongodb
